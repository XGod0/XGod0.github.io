工具篇之Empire

Empire是一个PowerShell后渗透框架，基于密码安全通信和灵活的架构。

一、常用命令

1.1 基础命令

命令 介绍
agenets 进入代理菜单
creds 往数据库添加或显示凭证
exit 结束Empire
help 显示帮助信息
interact 进入会话
list 列出存活的代理或监听器
listeners 进入监听器模块
load 加载Empire插件
plugins 列出可用插件
plugin 导入插件
preobfuscate 预处理Powershell模块
reload 重新导入模块
report 输出结果
reset 重置配置
resource 从文件中读取并执行Empire命令
searchmodule 查找模块
set 读取配置参数
show 显示配置
usemodule 使用模块
usestager 使用stager

1.2 控制命令

命令 介绍
clear 清除代理
dowmload PATH 下载文件
ps 显示进程信息
killdate 在某个时间删除代理
kill 删除某个进程
sleep interval [delay] 设置代理休眠
lostlimit 尝试几次连接
bypassuac 衍生出一个高权限的代理
info 显示信息
injectshellcode 往进程里注入shellcode
jobs 显示正在运行或结束的工作
kill 删除某代理或者进程
mimikatz 在客户端运行mimikatz
psinject 注入进程
pth 进行hash传递
revtoself 使用令牌或者凭证权限
sc 截屏
scriptcmd 在当前导入的powershell中执行函数
scriptimport 导入poweishell脚本并保留在内存中
shell 调用windows下的指令
sysinfo 查看系统信息
spawn 衍生出一个新的代理
steal_token 使用凭证或者令牌
updatecomms 更新监听器
updateprofile 更新传输策略
workinghours 设置工作时间
